# Front with React.JS

## Requirements

- [Docker](https://www.docker.com/)
- [Node JS](https://nodejs.org/en/download/)
  (Version recommended 14.0 or higher)
- [Yarn](https://yarnpkg.com/en/docs/install)

## To Run

- **yarn install**
- **yarn start**

Or

- **make build run**: Build and run docker image

Then Open the link <http://127.0.0.1:3000/>

## Other commands

- **yarn test**: Run tests
- **yarn test:lint**: Run linter and look up for code conventions.
- **yarn test:journey**: Run journey tests. (Requires of Backend service running)
- **yarn test:journey-open**: Run journey tests and opens cypress console
